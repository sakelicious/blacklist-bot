# Blacklist Bot
Proiect gata configurat pentru a rezolva eroarea de deployment `ENOENT: no such file or directory, scandir './commands'`.

### Structură:
- `index.js` - Codul principal actualizat cu verificări automate pentru directoare.
- `commands/` - Folderul în care îți pui toate comenzile de tip slash. Conține deja o comandă de test (`ping.js`).
- `.env.example` - Model pentru fișierul de configurare. Rezeamă-l în `.env` local și pune tokenul tău.

### Deployment:
Asigură-te că urci tot conținutul pe GitHub (inclusiv folderul `commands`).
